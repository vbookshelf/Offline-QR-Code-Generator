# Offline QR Code Generator
A browser based local desktop tool to quickly create static QR codes and download them in PNG or SVG formats. Built for graphic designers, small businesses and non-profits who want a hassle free way to create QR codes.

xxxxxxxxx

- Istantly create QR codes for url's, WiFi and other text.
- Download and double-click the index.html file to launch from your desktop.
- Private and offline. No external server connections or tracking.
- Export codes as PNG or SVG.
- Supports transparent backgrounds.
- Works forever.
- Free and open source.
